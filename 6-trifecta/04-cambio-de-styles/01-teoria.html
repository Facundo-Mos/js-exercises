# Cambiar estilos CSS

¡No solo podemos usar los métodos del DOM para manipular nuestro contenido HTML, sino que también podemos cambiar nuestros estilos CSS!

Al seleccionar un elemento, puedes acceder a su propiedad .style.

document.getElementById("posted").style;

A partir de ahí, puedes usar cualquiera de las propiedades CSS comunes que ya conoces y te encantan. Aquí tienes algunas de las más comunes:

Propiedad CSS Propiedad DOM
background-color backgroundColor
border border
border-radius borderRadius
font-size fontSize
margin margin
padding padding
text-align textAlign

Importante: En el DOM, las propiedades CSS de varias palabras están en formato camelCase.

Puedes cambiar el estilo de un elemento completamente desde JavaScript:

document.getElementById("posted").style.textAlign = "center";
Instrucciones

Los anillos que cambian de color según el estado de ánimo estuvieron muy de moda en la década de 1970. Una "piedra lunar" especial cambiaba de color según la temperatura del dedo, ¡y la idea era que uno pudiera saber cómo se sentía en general según ese color!

¡Practiquemos cambiando estilos CSS con la propiedad .style creando un anillo que cambie de color según el estado de ánimo!

Comienza con el siguiente código HTML:

<!DOCTYPE html>
<html lang="en">
<head>
    <link rel="stylesheet" href="styles.css">
    <title>Mood Ring</title>
</head>
<body>
    <div class="mood-ring" id="moodRing">
        <div class="stone" id="stone"></div>
    </div>
    <h1>How Are You Feeling?</h1>
    <script src="script.js"></script>
</body>
</html>

A continuación, añade el siguiente código CSS:

body {
    text-align: center;
}

.mood-ring {
    width: 150px;
    height: 150px;
    margin: 20px auto;
    border: 15px solid #ccc;
    border-radius: 50%;
    position: relative;
}

.stone {
    width: 125px;
    height: 125px;
    background-color: #3498db; /* Default color */
    border-radius: 50%;
    position: absolute;
    top: 50%;
    left: 50%;
    transform: translate(-50%, -50%);
    transition: background-color 0.5s ease-in-out;
}

Dentro de un archivo script.js, crea una variable que seleccione el elemento con el id de "stone".

A continuación, crea una variable randomNumber y usa Math.floor() y Math.random() para generar un número aleatorio entre 1 y 9.

Luego, utiliza una estructura condicional if, else if, y else para cambiar la propiedad .style.backgroundColor según el valor de tu randomNumber.
