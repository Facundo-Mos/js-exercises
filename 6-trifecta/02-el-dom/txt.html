El DOM

El HTML de un sitio web puede considerarse como un árbol. Dentro del archivo del documento, se encuentra el elemento raíz <html>. De este elemento raíz surgen las dos ramas principales: los elementos <head> y <body>.

Esto se puede visualizar como un árbol al revés, como se muestra a continuación:

Esto se conoce como Modelo de Objetos de Documento o DOM. Desde la perspectiva del navegador, el DOM es una familia de árboles que representan cómo se organizan y relacionan los elementos HTML de un sitio.

El objeto del documento

Al visitar un sitio, se crea un nuevo árbol DOM. Cada elemento bajo los elementos <head> y <body> es como una fruta que cuelga del árbol.

¡Con JavaScript, podemos acceder a cualquiera de estos elementos con un objeto especial document!

Suponiendo que el siguiente código está vinculado con un archivo HTML:

console.log(document);

¡Todo lo que se encuentre en ese archivo HTML se registrará en la consola! 🤯

.querySelector()

Descubrirás que document es muy potente. Con él, puedes seleccionar cualquier elemento del HTML mediante métodos. Uno de ellos es .querySelector().

// Select by element name
document.querySelector("p");

// Select by class name
document.querySelector(".class-name");

// Select by id name
document.querySelector("#id-name");

Se pasa una cadena al método .querySelector() que representa un elemento y se devuelve la primera instancia de dicho elemento. Esta cadena puede ser el nombre del elemento, un nombre de class o un nombre de id.

.innerHTML

No solo puedes acceder a los elementos con métodos DOM, sino que también puedes acceder al contenido de esos elementos con propiedades como .innerHTML:

document.querySelector("div").innerHTML;
// <p>Howdy!</p>
