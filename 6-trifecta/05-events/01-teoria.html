# Eventos

Hasta ahora, solo hemos modificado el HTML generado cambiando el código y guardando el archivo. Pero la mayoría de los sitios son mucho más interactivos; puedes interactuar con ellos para realizar cambios al instante. Esto podría incluir:

  • Hacer clic en un botón del sitio.
  • Pasar el ratón por encima de algo.
  • Rellenar y enviar un formulario.
  • Utilizar una de las teclas de tu teclado.

Todos estos son ejemplos de eventos.

Si bien HTML es donde pueden tener lugar los eventos, JavaScript es donde los configuramos. Esto se hace con un método del DOM llamado .addEventListener().

Al seleccionar un elemento, puedes utilizar .addEventListener() para ejecutar una función en ese elemento si el usuario realiza una acción específica, como hacer clic o pasar el cursor por encima:

let likeButton = document.getElementById("like-button");
let likeCount = document.getElementById("like-count");

let numberOfLikes = 0;

likeButton.addEventListener("click", function() {
  numberOfLikes++;
  likeCount.innerText = numberOfLikes;
});

En el ejemplo anterior, estamos usando .addEventListener() para adjuntar un evento "click" al elemento seleccionado likeButton. Cuando se hace clic, una función actualiza el .innerText del elemento seleccionado likeCount y lo incrementa en 1, a través de la variable numberOfLikes.

Instrucciones

Vamos a usar detectores de eventos para crear un generador de citas sencillo. Estas citas tendrán como tema la meditación y la respiración. 🧘

Agrega lo siguiente a un archivo index.html:

<!DOCTYPE html>
<html lang="en">
<head>
  <link href="styles.css" rel="stylesheet" />
  <title>Chill Pill</title>
</head>
<body>
  <div id="wrapper">
    <h1>Quote Generator</h1>
    <h2>Relax and Take a Chill Pill</h2>
    <div id="quote-div">
      <p id="quote-text"></p>
    </div>
    <button id="quote-button">Generate Quote</button>
  </div>
  <script src="script.js"></script>
</body>
</html>

A continuación, añade lo siguiente a un archivo styles.css:

#wrapper {
  border: 2px solid black;
  border-radius: 5px;
  width: 50%;
  margin: 25vh auto;
  padding: 10px;
  text-align: center;
  transition: background-color 0.5s ease-in-out;
}

#quote-div {
  border: 1px solid black;
  border-radius: 5px;
  width: 75%;
  margin: 10px auto;
  background-color: #fff;
}

Para el código JavaScript, comencemos copiando y pegando lo siguiente en el archivo script.js:

const quoteList = [
  "Some doors only open from the inside. Breath is a way of accessing that door.",
  "What has to be taught first, is the breath.",
  "Remember the blue sky. It may at times be obscured by clouds, but it is always there.",
  "In the midst of movement and chaos, keep stillness inside of you.",
  "We can't control the sea, but we can learn how to surf the waves.",
  "Feelings come and go like clouds in a windy sky. Conscious breathing is my anchor.",
  "To understand the immeasurable, the mind must be extraordinarily quiet, still."
];

const colors = ["#e81416", "#ffa500", "#faeb36", "#79c314", "#487de7", "#4b369d", "#70369d"];

A continuación, utiliza los métodos de selección del DOM para crear las siguientes variables:

  • Una variable wrapperDiv que seleccione el elemento <div> con un valor de id de "wrapper".
  • Una variable quoteText que seleccione el elemento de párrafo <p> con un valor de id de "quote-text".
  • Una variable quoteButton que seleccione el elemento <button> con un valor de id de "quote-button".

¡Ahora es el momento de añadir el evento!

Utiliza el método .addEventListener() en la variable quoteButton. Pasa la cadena "click" junto con una función que haga lo siguiente:

  • Cree un randomIndex a partir del array quoteList, usando Math.floor() y Math.random().
  • Utilice este randomIndex para obtener una cita aleatoria del array quoteList y guardarla en la variable randomQuote.
  • Cambie el texto de la variable quoteText por el valor de la variable randomQuote.
  • Utilice randomIndex nuevamente para cambiar el color de fondo de la variable wrapperDiv a alguno del array colors.
