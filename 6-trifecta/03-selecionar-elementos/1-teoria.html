41. Bling de línea directa

Seleccionar elementos HTML

¡Es hora de entrar en la parte divertida del DOM!

El objeto document viene con métodos y propiedades que podemos usar para seleccionar uno o más elementos HTML.

Dos de los métodos más utilizados son los siguientes:

  • .getElementById()
  • .getElementsByClassName()

Todos estos métodos aceptan cadenas como parámetros, que representan un elemento que puede existir en un archivo HTML conectado.

Por ejemplo:

<div id="status-update" class="post">
  <p>¡Acabo de aterrizar de mi viaje en avión! ✈️</p>
</div>

<div id="posted" class="post">
  <img src="https://cdn.pixabay.com/photo/2017/05/12/13/17/night-flight-2307018_640.jpg" alt="avión despegando de la pista" />
  <p>¡Estoy a punto de despegar!</p>
  <p>¡Qué emoción! 😅</p>
</div>

.getElementsByClassName()

Puede seleccionar varios elementos si comparten el mismo atributo class, utilizando el método .getElementsByClassName():

console.log(document.getElementsByClassName("post"));
// Resultado: HTMLCollection(2) [div#status-update.post, div#posted.post, status-update: div#status-update.post, posted: div#posted.post]

Espera, ¿qué devolvió esto? Un HTMLCollection? Este objeto es similar a un array. Puedes acceder a un elemento individual de este objeto mediante índices:

console.log(document.getElementsByClassName("post")[0]);
// Resultado: <div id="status-update" class="post"><p>¡Acabo de aterrizar de mi viaje en avión! ✈️</p></div>

.getElementById()

También puede seleccionar un elemento con un atributo id coincidente, utilizando el método .getElementById():

console.log(document.getElementById("posted"));
/* Resultado:
<div id="posted" class="post">
  <p>¡Estoy a punto de despegar!</p>
  <p>¡Qué emoción! 😅</p>
</div>
*/

Por último, puedes usar una combinación de estos métodos si es posible. Por ejemplo, si quieres seleccionar el primer elemento <p> del segundo <div> (con un id de "posted"), una forma de hacerlo es con .getElementById("posted") seguido de .querySelector("p"):

document.getElementById("posted").querySelector("p");
// Resultado: <p>¡Estoy a punto de despegar!</p>

Actualización de elementos HTML

Recuerda: algunos objetos contienen ciertas propiedades y métodos. Esto incluye los objetos de elemento devueltos por los métodos del objeto document.

Dependiendo del elemento que seleccionemos, podremos acceder a su contenido con una de las siguientes propiedades:

  • .innerText
  • .src

.innerText (texto interno)

La propiedad .innerText devuelve el texto representado dentro de un elemento seleccionado, incluidas todas las nuevas líneas.

console.log(document.getElementById("posted").innerText);
// Resultado: "¡Estoy a punto de despegar!\n\n¡Qué emoción! 😅"

También podemos cambiar el texto de un elemento seleccionado asignándole un nuevo valor de cadena con la propiedad .innerText:

let newText = document.getElementById("posted").querySelector("p").innerText = "¡Estoy a punto de despegar! Es un vuelo temprano, ¡así que estoy listo para dormir! 😴";

console.log(newText);
// Resultado: ¡Estoy a punto de despegar! Es un vuelo temprano, ¡así que estoy listo para dormir! 😴

.src

Si trabajas con un elemento de imagen <img>, puedes acceder a su atributo src con... ¡lo adivinaste! ¡La propiedad .src!

console.log(document.querySelector("img").src);
// Resultado: 'https://cdn.pixabay.com/photo/2017/05/12/13/17/night-flight-2307018_640.jpg'

Y para cambiar la fuente de una imagen, supongamos que tenemos una cara triste que usa esto:

console.log(document.querySelector("img").src);
// Resultado: 'smiley1.png'
☹️

Podemos actualizar la propiedad .src de la imagen asignando una nueva cadena:

document.querySelector("img").src = "smiley2.png";
