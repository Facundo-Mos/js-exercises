# ¡Felicitaciones!

¡Has llegado al final de la Trilogía de los Orígenes! 🥹

¡Has experimentado cómo los tres lenguajes (HTML, CSS y JavaScript) trabajan juntos para crear un sitio web interactivo! Has adquirido todas las herramientas necesarias para comenzar tu andadura en el desarrollo web.

En este capítulo, aprendiste lo siguiente:

  • Cómo el elemento <script> nos permite conectar nuestro JavaScript con nuestro HTML.
  • El DOM y cómo podemos acceder a los elementos de un sitio web con el objeto document de JavaScript.
  • Los elementos se pueden seleccionar a través del objeto document con métodos como:
    • .querySelector()
    • .getElementById()
    • .getElementsByClassName()
  • Podemos actualizar dinámicamente nuestros elementos seleccionados con propiedades como .innerHTML, .innerText, y .src (por ejemplo, para imágenes).
  • La propiedad .style puede cambiar el CSS de los elementos seleccionados, como el color de fondo (es decir, .style.backgroundColor).
  • Mediante el uso de eventos, incluso podemos hacer que una función de JavaScript se ejecute solo cuando el usuario realiza una acción específica (por ejemplo, al hacer clic en un botón con el ratón).
Instrucciones

¡Es hora de juntarlo todo y crear algunos memes!

Los memes de Internet son una forma divertida de expresar pensamientos, sentimientos e ideas con una imagen y unas pocas palabras.

Comencemos con el HTML:

<!DOCTYPE html>
    <html lang="en">
    <head>
    <link href="styles.css" rel="stylesheet" />
    <title>Cap That</title>
    </head>
    <body>
    <h1>Welcome to the Meme Machine</h1>
    <button id="generator-button">Generate Meme</button>
    <div id="meme-wrapper">
        <img width="600" id="random-meme" src="" />
        <h2 id="random-caption"></h2>
    </div>
    <script src="script.js"></script>
    </body>
    </html>

A continuación, añadiremos el siguiente código CSS:

body {
    text-align: center;
    }

    #generator-button {
    padding: 10px;
    border: 2px solid black;
    border-radius: 5px;
    font-weight: bolder;
    font-size: xx-large;
    }

    #meme-wrapper {
    border: 4px solid black;
    width: 700px;
    height: 500px;
    padding: 5px;
    margin: 15px auto;
    }

    #random-meme {
    border-radius: 5px;
    }

    #random-caption {
    margin: 0;
}

En un archivo script.js, copia y pega el siguiente array:

const memeArray = [
    "https://i.imgur.com/bSi4xLb.png",
    "https://i.imgur.com/6y0G7N0.png",
    "https://i.imgur.com/LXnRao1.png",
    "https://i.imgur.com/Qqoxh1N.png"
];

Nota: ¡Si lo deseas, puedes añadir más imágenes!

¡Luego, crea un captionsArray con tantas cadenas de texto ingeniosas como desees!

A continuación, te explicamos lo que debes hacer mediante los métodos y propiedades del DOM:

  1. Selecciona el elemento de imagen <img> con el valor de id igual a "random-meme" y guárdalo en una variable.
  2. Selecciona el elemento de encabezado <h2> con el valor de id igual a "random-caption" y guárdalo en una variable.
  3. Selecciona el elemento <button> con el valor de id igual a "generator-button" y guárdalo en una variable.

A continuación, añade un detector de eventos (event listener) de tipo "click" al elemento "generator-button" seleccionado, y realiza lo siguiente dentro de la función:

  • Genera 2 números de índice aleatorios: uno entre 0 y la longitud de memeArray, y otro entre 0 y la longitud de captionsArray.
  • Utiliza uno de los números aleatorios para cambiar la propiedad .src del elemento de imagen "random-meme" por una de las imágenes del memeArray.
  • Utiliza el otro número aleatorio para cambiar la propiedad .innerText del elemento de encabezado "random-caption" por una de las cadenas del captionsArray.

¡Guarda el archivo y prueba tu máquina de memes! Al hacer clic en el botón, aparecerá una imagen y un texto aleatorios.
